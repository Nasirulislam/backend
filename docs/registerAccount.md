# Register account

Create a new account with the given parameters

**URL**

  /api/accounts

**Method:**
  
  `POST`
  
**Parameters**

**Content-Type**

  `application/json`

**Success Response:**
  
- Account succesfully created

```json
{
    "account": {
        "username": "username",
        "email": "user@gmail.com",
        "id": 4
    }
}
```
 
**Error Response:**

Not documented
