**Error Codes**
----

| Code | Error                                   |
|------|-----------------------------------------|
| 1    | Missing item identifier                 |
| 2    | Incorrect item identifier               |
| 3    | Unauthorized                            |
| 4    | Forbidden                               |
| 5    | Invalid item identifier                 |
| 6    | Missing title                           |
| 7    | Missing description                     |
| 8    | Invalid title                           |
| 9    | Invalid description                     |
