# Fetch image

**URL**

  /api/images/[image_id]

**Method:**
  
  `GET`
  
**Parameters**

- image_id: The id of the image to fetch

**Content-Type**

  `application/json`

**Success Response:**
  
- Imagen with the given id

 
**Error Response:**

- Invalid image identifier

```json
{
    "code": 12
}
```
